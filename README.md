# ELMO, embeddable scripting language for Go

## quick start

```bash
go get github.com/okke/elmo/tools/elmo
go install github.com/okke/elmo/tools/elmo
elmo
```

and you should see:

```
e>mo: 
```

to see all basic commands type:

```elmo
help
```

to get help on specific command:
```elmo
help puts
```
